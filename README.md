# dotfiles

dotfiles for my CachyOS/Niri/DMS setup

This is a work in progress as I slowly make my way through ricing CachyOS

## App dotfiles

- chezmoi (Managing the dotfiles)
- yazi (terminal file manager, includes dotfile helpers)
- mutagen (consistent colour theming)
- mpd (music server)
- rmpc (music player front end)

# Philosophy

- Perfect is the enemy of done
- Iteratively make things better
- Start with some defaults until they annoy me

# Goals

## Themes

- Use matugen templates for theme colours
- If possible, reference the colours config in the theme config

## Keyboard shortcuts

- Sortcuts shall be consistant
- Shortcuts shall use modifier keys to differentiate between modes

### Context

- WM shortcuts shall be prefixed with `Super`
- App meta/menu shortcuts shall be prefixed with `Alt`
- App command/action shortcuts shall be prefixed with `Ctrl`

### Keybindings

These should apply to all contexts (i hope!)

|----------|----------|
| Shortcut | Function |
|----------|----------|
| q        | Quit     |
| s        | Save     |
